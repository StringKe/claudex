## 安裝方式

<Tabs>
  <TabItem label="一鍵安裝" icon="terminal">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
    安裝腳本自動偵測 OS 和架構，從 GitHub 下載最新 release，安裝到 `~/.local/bin/claudex`。

    自訂安裝目錄：
    ```bash
    CLAUDEX_INSTALL_DIR=/usr/local/bin curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
  </TabItem>
  <TabItem label="Cargo" icon="seti:rust">
    ```bash
    cargo install --git https://github.com/StringKe/claudex
    ```
    需要 Rust 工具鏈。二進位檔案安裝到 `~/.cargo/bin/claudex`。
  </TabItem>
  <TabItem label="GitHub Releases" icon="github">
    從 [GitHub Releases](https://github.com/StringKe/claudex/releases) 下載適合您平台的預編譯二進位檔案。

    可用目標：
    - `x86_64-unknown-linux-gnu`
    - `x86_64-unknown-linux-musl`
    - `aarch64-unknown-linux-gnu`
    - `aarch64-unknown-linux-musl`
    - `x86_64-apple-darwin`
    - `aarch64-apple-darwin`
    - `x86_64-pc-windows-msvc`

    解壓並放入 PATH：
    ```bash
    tar xzf claudex-v*.tar.gz
    chmod +x claudex
    mv claudex ~/.local/bin/
    ```
  </TabItem>
</Tabs>

## 系統需求

| 需求 | 說明 |
|------|------|
| **作業系統** | macOS（Intel / Apple Silicon）、Linux（x86_64 / ARM64）、Windows（x86_64） |
| **Claude Code** | 必須已安裝，且 `claude` 指令在 PATH 中可用 |
| **選用** | [Ollama](https://ollama.com) 用於智慧路由和上下文引擎功能 |

## 驗證安裝

```bash
claudex --version
```
