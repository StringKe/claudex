---
title: TUI 儀表板
description: 用於監控和管理 profile 的即時終端儀表板
---

TUI 儀表板提供即時終端介面，用於監控 profile 健康狀態、檢視指標，以及快速啟動 Claude Code 工作階段。

## 啟動

```bash
# 明確啟動
claudex dashboard

# 自動啟動（存在 profile 且未指定子指令時）
claudex
```

## 鍵盤快捷鍵

| 按鍵 | 動作 |
|------|------|
| `j` / `↓` | 向下移動選取 |
| `k` / `↑` | 向上移動選取 |
| `Enter` | 以選取的 profile 啟動 Claude Code |
| `t` | 測試選取 profile 的連線 |
| `a` | 新增 profile（顯示提示） |
| `d` | 刪除選取的 profile（顯示提示） |
| `p` | 切換代理啟動/停止 |
| `/` | 進入搜尋模式 |
| `Esc` | 退出搜尋模式 / 關閉說明 |
| `?` | 顯示說明彈出視窗 |
| `q` / `Ctrl+C` | 退出儀表板 |

## 儀表板版面配置

儀表板顯示：

- **Profile 清單** — 所有已設定的 profile，含名稱、模型、提供商類型和健康狀態
- **健康指標** — 色彩標示的狀態（綠色 = 正常、黃色 = 降級、紅色 = 斷路器開啟）
- **指標面板** — 每個 profile 的請求數、成功率、平均延遲
- **日誌面板** — 即時捲動的日誌輸出

## 健康狀態

儀表板執行背景健康檢查器，每約 30 秒對所有已啟用的 profile 進行探測。健康狀態即時反映：

| 顏色 | 含義 |
|------|------|
| 綠色 | 提供商回應正常 |
| 黃色 | 偵測到近期失敗 |
| 紅色 | 斷路器已開啟 |
| 灰色 | Profile 已停用 |
