## 支援的提供商

| 類別 | 提供商 |
|------|--------|
| **Direct Anthropic** | Anthropic、MiniMax |
| **OpenAI Compatible** | Grok (xAI)、OpenAI、DeepSeek、Kimi/Moonshot、GLM (Zhipu)、OpenRouter、Groq、Mistral、Together AI、Perplexity、Cerebras |
| **雲端平台** | Azure OpenAI、Google Vertex AI |
| **OAuth 訂閱** | Claude Max/Pro、ChatGPT Plus/Pro (Codex)、Google Gemini、GitHub Copilot、GitLab Duo、Qwen、Kimi |
| **本地模型** | Ollama、vLLM、LM Studio、llama.cpp |
| **透過 LiteLLM** | AWS Bedrock 及任何 LiteLLM 支援的提供商 |

## 快速安裝

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## 運作原理

```
claudex run grok
    │
    ├── Start proxy → 127.0.0.1:13456
    │
    └── exec claude with:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

代理透明地在 Anthropic 和 OpenAI 協定之間進行翻譯，讓 Claude Code 與任何支援的提供商無縫協作。使用 OAuth profile 時，設定 `ANTHROPIC_AUTH_TOKEN`（而非 `ANTHROPIC_API_KEY`）以避免與 Claude 自身訂閱登入機制衝突。
