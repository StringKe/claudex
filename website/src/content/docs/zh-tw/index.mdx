<CardGrid stagger>

  </Card>
  <Card title="智慧路由" icon="random">

  </Card>

  <Card title="TUI 儀表板" icon="laptop">
    即時終端儀表板，顯示 profile 健康狀態、請求指標、日誌和快速啟動。完整鍵盤導覽支援。
  </Card>
  <Card title="自動更新" icon="up-caret">

## 快速安裝

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## 運作原理

```
claudex run grok
    │
    ├── 啟動代理 → 127.0.0.1:13456
    │
    └── 執行 claude，設定：
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

代理透明地在 Anthropic 和 OpenAI 協定之間進行翻譯，讓 Claude Code 與任何支援的提供商無縫協作。使用 OAuth profile 時，設定 `ANTHROPIC_AUTH_TOKEN`（而非 `ANTHROPIC_API_KEY`）以避免與 Claude 自身訂閱登入衝突。
