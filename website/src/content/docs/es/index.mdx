## Instalacion Rapida

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## Como Funciona

```
claudex run grok
    │
    ├── Start proxy → 127.0.0.1:13456
    │
    └── exec claude with:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

El proxy traduce de forma transparente entre los protocolos Anthropic y OpenAI, de modo que Claude Code funciona sin problemas con cualquier proveedor compatible. Al usar perfiles OAuth, se establece `ANTHROPIC_AUTH_TOKEN` (no `ANTHROPIC_API_KEY`) para evitar conflictos con el inicio de sesion por suscripcion propio de Claude.
