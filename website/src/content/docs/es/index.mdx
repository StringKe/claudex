## Instalacion rapida

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## Como funciona

```
claudex run grok
    │
    ├── Start proxy → 127.0.0.1:13456
    │
    └── exec claude with:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

El proxy traduce de forma transparente entre los protocolos Anthropic y OpenAI, permitiendo que Claude Code funcione sin problemas con cualquier proveedor soportado. Al usar perfiles OAuth, se establece `ANTHROPIC_AUTH_TOKEN` (y no `ANTHROPIC_API_KEY`) para evitar conflictos con el mecanismo de inicio de sesion por suscripcion propio de Claude.
