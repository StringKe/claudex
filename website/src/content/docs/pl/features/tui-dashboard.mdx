---

Pulpit TUI zapewnia terminalowy interfejs czasu rzeczywistego do monitorowania kondycji profili, przegladania metryk i szybkiego uruchamiania sesji Claude Code.

## Uruchomienie

```bash
# Jawne uruchomienie
claudex dashboard

# Automatyczne uruchomienie (gdy profile istnieja i nie podano podpolecenia)
claudex
```

## Skroty klawiaturowe

| Klawisz | Akcja |
|---------|-------|
| `j` / `↓` | Przesun zaznaczenie w dol |
| `k` / `↑` | Przesun zaznaczenie w gore |
| `Enter` | Uruchom Claude Code z wybranym profilem |
| `t` | Przetestuj lacznosc dla wybranego profilu |
| `a` | Dodaj nowy profil (pokazuje podpowiedz) |
| `d` | Usun wybrany profil (pokazuje podpowiedz) |
| `p` | Przelacz uruchomienie/zatrzymanie proxy |
| `/` | Wejdz w tryb wyszukiwania |
| `Esc` | Wyjdz z trybu wyszukiwania / zamknij pomoc |
| `?` | Pokaz wyskakujace okno pomocy |
| `q` / `Ctrl+C` | Zamknij pulpit |

## Uklad pulpitu

Pulpit wyswietla:

- **Lista profili** — Wszystkie skonfigurowane profile z nazwa, modelem, typem dostawcy i statusem kondycji
- **Wskazniki kondycji** — Status kodowany kolorami (zielony = zdrowy, zolty = zdegradowany, czerwony = wylacznik automatyczny otwarty)
- **Panel metryk** — Liczba zadan per profil, wskaznik sukcesu, srednie opoznienie
- **Panel logow** — Wyjscie logow przewijane w czasie rzeczywistym

## Status kondycji

Pulpit uruchamia sprawdzanie kondycji w tle, ktore pinguje wszystkie aktywne profile co ~30 sekund. Status kondycji jest odzwierciedlany w czasie rzeczywistym:

| Kolor | Znaczenie |
|-------|-----------|
| Zielony | Dostawca odpowiada normalnie |
| Zolty | Wykryto ostatnie awarie |
| Czerwony | Wylacznik automatyczny jest otwarty |
| Szary | Profil jest wylaczony |
