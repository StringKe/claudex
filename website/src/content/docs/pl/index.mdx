## Szybka instalacja

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## Jak to działa

```
claudex run grok
    │
    ├── Start proxy → 127.0.0.1:13456
    │
    └── exec claude with:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

Proxy transparentnie tłumaczy między protokołami Anthropic i OpenAI, dzięki czemu Claude Code działa bezproblemowo z dowolnym obsługiwanym dostawcą. Przy profilach OAuth ustawiany jest `ANTHROPIC_AUTH_TOKEN` (nie `ANTHROPIC_API_KEY`), aby uniknąć konfliktów z własnym logowaniem subskrypcyjnym Claude.
