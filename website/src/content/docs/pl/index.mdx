## Szybka instalacja

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## Jak to dziala

```
claudex run grok
    │
    ├── Start proxy → 127.0.0.1:13456
    │
    └── exec claude with:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

Proxy transparentnie tlumaczy miedzy protokolami Anthropic i OpenAI, dzieki czemu Claude Code dziala bezproblemowo z dowolnym obslugiwanym dostawca. Przy profilach OAuth ustawiany jest `ANTHROPIC_AUTH_TOKEN` (nie `ANTHROPIC_API_KEY`), aby uniknac konfliktow z wlasnym logowaniem subskrypcyjnym Claude.
