## 빠른 설치

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## 작동 방식

```
claudex run grok
    │
    ├── 프록시 시작 → 127.0.0.1:13456
    │
    └── 다음 설정으로 claude 실행:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

프록시는 Anthropic과 OpenAI 프로토콜 간을 투명하게 변환하여, Claude Code가 지원되는 모든 제공자와 원활하게 작동합니다. OAuth 프로파일을 사용할 때는 Claude 자체 구독 로그인과의 충돌을 피하기 위해 `ANTHROPIC_API_KEY` 대신 `ANTHROPIC_AUTH_TOKEN`이 설정됩니다.
