TUI 대시보드는 프로파일 상태 모니터링, 지표 확인, Claude Code 세션 빠른 실행을 위한 실시간 터미널 인터페이스를 제공합니다.

## 실행

```bash
# 명시적으로 실행
claudex dashboard

# 자동 실행 (프로파일이 존재하고 서브 명령 없이 실행 시)
claudex
```

## 키보드 단축키

| 키 | 동작 |
|-----|--------|
| `j` / `↓` | 선택 아래로 이동 |
| `k` / `↑` | 선택 위로 이동 |
| `Enter` | 선택된 프로파일로 Claude Code 실행 |
| `t` | 선택된 프로파일 연결 테스트 |
| `a` | 새 프로파일 추가 (힌트 표시) |
| `d` | 선택된 프로파일 삭제 (힌트 표시) |
| `p` | 프록시 시작/중지 전환 |
| `/` | 검색 모드 진입 |
| `Esc` | 검색 모드 종료 / 도움말 닫기 |
| `?` | 도움말 팝업 표시 |
| `q` / `Ctrl+C` | 대시보드 종료 |

## 대시보드 레이아웃

대시보드에 표시되는 내용:

- **프로파일 목록** — 이름, 모델, 제공자 유형, 상태를 포함한 모든 설정된 프로파일
- **상태 표시기** — 색상 코드 상태 (초록색 = 정상, 노란색 = 저하됨, 빨간색 = 서킷 브레이커 열림)
- **지표 패널** — 프로파일별 요청 수, 성공률, 평균 지연 시간
- **로그 패널** — 실시간 스크롤 로그 출력

## 상태 표시

대시보드는 약 30초마다 활성화된 모든 프로파일을 핑하는 백그라운드 상태 확인기를 실행합니다. 상태가 실시간으로 반영됩니다:

| 색상 | 의미 |
|-------|---------|
| 초록색 | 제공자가 정상적으로 응답 중 |
| 노란색 | 최근 실패 감지됨 |
| 빨간색 | 서킷 브레이커 열림 |
| 회색 | 프로파일 비활성화됨 |
