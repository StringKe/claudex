---
title: TUI 仪表盘
description: 实时终端仪表盘，用于监控和管理 profile
---

TUI 仪表盘提供实时终端界面，用于监控 profile 健康状态、查看指标和快速启动 Claude Code 会话。

## 启动

```bash
# 显式启动
claudex dashboard

# 自动启动（有 profile 且未指定子命令时）
claudex
```

## 快捷键

| 按键 | 操作 |
|------|------|
| `j` / `↓` | 向下选择 |
| `k` / `↑` | 向上选择 |
| `Enter` | 使用选定 profile 启动 Claude Code |
| `t` | 测试选定 profile 的连通性 |
| `a` | 添加新 profile（显示提示） |
| `d` | 删除选定 profile（显示提示） |
| `p` | 切换代理启动/停止 |
| `/` | 进入搜索模式 |
| `Esc` | 退出搜索模式 / 关闭帮助 |
| `?` | 显示帮助弹窗 |
| `q` / `Ctrl+C` | 退出仪表盘 |

## 仪表盘布局

仪表盘显示：

- **Profile 列表** — 所有已配置的 profile，包含名称、模型、提供商类型和健康状态
- **健康指示器** — 颜色编码状态（绿色 = 健康，黄色 = 降级，红色 = 断路器已打开）
- **指标面板** — 每个 profile 的请求数、成功率、平均延迟
- **日志面板** — 实时滚动日志输出

## 健康状态

仪表盘运行后台健康检查器，每 ~30 秒 ping 所有启用的 profile。健康状态实时反映：

| 颜色 | 含义 |
|------|------|
| 绿色 | 提供商正常响应 |
| 黄色 | 检测到近期故障 |
| 红色 | 断路器已打开 |
| 灰色 | Profile 已禁用 |
