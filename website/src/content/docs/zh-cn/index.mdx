## 支持的提供商

| 类别 | 提供商 |
|------|--------|
| **原生 Anthropic** | Anthropic、MiniMax |
| **OpenAI 兼容** | Grok (xAI)、OpenAI、DeepSeek、Kimi/Moonshot、GLM (智谱)、OpenRouter、Groq、Mistral、Together AI、Perplexity、Cerebras |
| **云平台** | Azure OpenAI、Google Vertex AI |
| **OAuth 订阅** | Claude Max/Pro、ChatGPT Plus/Pro (Codex)、Google Gemini、GitHub Copilot、GitLab Duo、Qwen、Kimi |
| **本地模型** | Ollama、vLLM、LM Studio、llama.cpp |
| **通过 LiteLLM** | AWS Bedrock，以及任何 LiteLLM 支持的提供商 |

## 快速安装

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## 工作原理

```
claudex run grok
    │
    ├── Start proxy → 127.0.0.1:13456
    │
    └── exec claude with:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

代理在 Anthropic 和 OpenAI 协议之间透明翻译，让 Claude Code 与任何支持的提供商无缝协作。使用 OAuth profile 时，设置 `ANTHROPIC_AUTH_TOKEN`（非 `ANTHROPIC_API_KEY`）以避免与 Claude 自身的订阅登录机制冲突。
