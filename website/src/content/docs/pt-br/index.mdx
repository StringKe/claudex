## Instalacao Rapida

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## Como Funciona

```
claudex run grok
    │
    ├── Inicia proxy → 127.0.0.1:13456
    │
    └── executa claude com:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

O proxy traduz de forma transparente entre os protocolos Anthropic e OpenAI, para que o Claude Code funcione perfeitamente com qualquer provedor suportado. Ao usar perfis OAuth, `ANTHROPIC_AUTH_TOKEN` (nao `ANTHROPIC_API_KEY`) e definido para evitar conflitos com o proprio login de assinatura do Claude.
