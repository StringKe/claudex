## Metodos de Instalacao

<Tabs>
  <TabItem label="Linha unica" icon="terminal">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
    O script de instalacao detecta automaticamente seu sistema operacional e arquitetura, baixa a versao mais recente do GitHub e instala em `~/.local/bin/claudex`.

    Substituir o diretorio de instalacao:
    ```bash
    CLAUDEX_INSTALL_DIR=/usr/local/bin curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
  </TabItem>
  <TabItem label="Cargo" icon="seti:rust">
    ```bash
    cargo install --git https://github.com/StringKe/claudex
    ```
    Requer a toolchain do Rust. O binario sera instalado em `~/.cargo/bin/claudex`.
  </TabItem>
  <TabItem label="GitHub Releases" icon="github">
    Baixe o binario pre-compilado para sua plataforma em [GitHub Releases](https://github.com/StringKe/claudex/releases).

    Alvos disponiveis:
    - `x86_64-unknown-linux-gnu`
    - `x86_64-unknown-linux-musl`
    - `aarch64-unknown-linux-gnu`
    - `aarch64-unknown-linux-musl`
    - `x86_64-apple-darwin`
    - `aarch64-apple-darwin`
    - `x86_64-pc-windows-msvc`

    Extraia e coloque o binario no seu `PATH`:
    ```bash
    tar xzf claudex-v*.tar.gz
    chmod +x claudex
    mv claudex ~/.local/bin/
    ```
  </TabItem>
</Tabs>

## Requisitos do Sistema

| Requisito | Detalhes |
|-----------|---------|
| **SO** | macOS (Intel / Apple Silicon), Linux (x86_64 / ARM64), Windows (x86_64) |
| **Claude Code** | Deve estar instalado e acessivel como `claude` no seu PATH |
| **Opcional** | [Ollama](https://ollama.com) para roteamento inteligente e funcionalidades do motor de contexto |

## Verificar Instalacao

```bash
claudex --version
```
