title: TUI-панель мониторинга
description: Панель мониторинга в реальном времени в терминале для управления профилями
---

TUI-панель мониторинга предоставляет интерфейс в терминале в режиме реального времени для наблюдения за состоянием профилей, просмотра метрик и быстрого запуска сессий Claude Code.

## Запуск

```bash
# Явный запуск
claudex dashboard

# Автозапуск (когда профили существуют и подкоманда не указана)
claudex
```

## Сочетания клавиш

| Клавиша | Действие |
|---------|----------|
| `j` / `↓` | Переместить выбор вниз |
| `k` / `↑` | Переместить выбор вверх |
| `Enter` | Запустить Claude Code с выбранным профилем |
| `t` | Тестировать подключение для выбранного профиля |
| `a` | Добавить новый профиль (показывает подсказку) |
| `d` | Удалить выбранный профиль (показывает подсказку) |
| `p` | Переключить запуск/остановку прокси |
| `/` | Войти в режим поиска |
| `Esc` | Выйти из режима поиска / закрыть справку |
| `?` | Показать всплывающую справку |
| `q` / `Ctrl+C` | Выйти из панели мониторинга |

## Структура панели

Панель мониторинга отображает:

- **Список профилей** — Все настроенные профили с именем, моделью, типом провайдера и статусом работоспособности
- **Индикаторы работоспособности** — Статус с цветовой кодировкой (зелёный = норма, жёлтый = ухудшение, красный = автоматический выключатель открыт)
- **Панель метрик** — Количество запросов, процент успеха, средняя задержка для каждого профиля
- **Панель логов** — Прокручиваемый вывод логов в реальном времени

## Статус работоспособности

Панель мониторинга запускает фоновую проверку работоспособности, которая опрашивает все активные профили каждые ~30 секунд. Статус работоспособности отражается в реальном времени:

| Цвет | Значение |
|------|----------|
| Зелёный | Провайдер отвечает нормально |
| Жёлтый | Обнаружены недавние сбои |
| Красный | Автоматический выключатель открыт |
| Серый | Профиль отключён |
