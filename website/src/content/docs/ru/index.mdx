## Быстрая установка

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## Как это работает

```
claudex run grok
    │
    ├── Start proxy → 127.0.0.1:13456
    │
    └── exec claude with:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

Прокси прозрачно переводит между протоколами Anthropic и OpenAI, поэтому Claude Code работает бесшовно с любым поддерживаемым провайдером. При использовании OAuth-профилей устанавливается `ANTHROPIC_AUTH_TOKEN` (а не `ANTHROPIC_API_KEY`), чтобы избежать конфликтов с собственным механизмом входа Claude по подписке, который внутренне использует `ANTHROPIC_API_KEY`.
