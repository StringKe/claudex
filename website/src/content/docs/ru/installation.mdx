## Способы установки

<Tabs>
  <TabItem label="Однострочник" icon="terminal">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
    Скрипт установки автоматически определяет вашу ОС и архитектуру, скачивает последний релиз с GitHub и устанавливает в `~/.local/bin/claudex`.

    Переопределение каталога установки:
    ```bash
    CLAUDEX_INSTALL_DIR=/usr/local/bin curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
  </TabItem>
  <TabItem label="Cargo" icon="seti:rust">
    ```bash
    cargo install --git https://github.com/StringKe/claudex
    ```
    Требуется набор инструментов Rust. Бинарный файл будет установлен в `~/.cargo/bin/claudex`.
  </TabItem>
  <TabItem label="GitHub Releases" icon="github">
    Скачайте предварительно собранный бинарный файл для вашей платформы из [GitHub Releases](https://github.com/StringKe/claudex/releases).

    Доступные цели:
    - `x86_64-unknown-linux-gnu`
    - `x86_64-unknown-linux-musl`
    - `aarch64-unknown-linux-gnu`
    - `aarch64-unknown-linux-musl`
    - `x86_64-apple-darwin`
    - `aarch64-apple-darwin`
    - `x86_64-pc-windows-msvc`

    Распакуйте и поместите бинарный файл в ваш `PATH`:
    ```bash
    tar xzf claudex-v*.tar.gz
    chmod +x claudex
    mv claudex ~/.local/bin/
    ```
  </TabItem>
</Tabs>

## Системные требования

| Требование | Подробности |
|------------|------------|
| **ОС** | macOS (Intel / Apple Silicon), Linux (x86_64 / ARM64), Windows (x86_64) |
| **Claude Code** | Должен быть установлен и доступен как `claude` в вашем PATH |
| **Опционально** | [Ollama](https://ollama.com) для функций умной маршрутизации и контекстного движка |

## Проверка установки

```bash
claudex --version
```
