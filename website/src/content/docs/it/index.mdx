## Installazione Rapida

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## Come Funziona

```
claudex run grok
    │
    ├── Avvia proxy → 127.0.0.1:13456
    │
    └── esegue claude con:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

Il proxy traduce in modo trasparente tra i protocolli Anthropic e OpenAI, così Claude Code funziona perfettamente con qualsiasi provider supportato. Quando si utilizzano profili OAuth, viene impostato `ANTHROPIC_AUTH_TOKEN` (non `ANTHROPIC_API_KEY`) per evitare conflitti con il login tramite abbonamento nativo di Claude.
