title: TUI-Dashboard
description: Echtzeit-Terminal-Dashboard zur Ueberwachung und Verwaltung von Profilen
---

Das TUI-Dashboard bietet eine Echtzeit-Terminal-Oberflaeche zur Ueberwachung des Profilzustands, Anzeige von Metriken und schnellen Start von Claude Code-Sitzungen.

## Starten

```bash
# Explizit starten
claudex dashboard

# Automatisch starten (wenn Profile vorhanden und kein Unterbefehl angegeben)
claudex
```

## Tastaturkuerzel

| Taste | Aktion |
|-------|--------|
| `j` / `↓` | Auswahl nach unten bewegen |
| `k` / `↑` | Auswahl nach oben bewegen |
| `Enter` | Claude Code mit ausgewaehltem Profil starten |
| `t` | Verbindung fuer ausgewaehltes Profil testen |
| `a` | Neues Profil hinzufuegen (zeigt Hinweis) |
| `d` | Ausgewaehltes Profil loeschen (zeigt Hinweis) |
| `p` | Proxy starten/stoppen umschalten |
| `/` | Suchmodus aktivieren |
| `Esc` | Suchmodus beenden / Hilfe schliessen |
| `?` | Hilfe-Popup anzeigen |
| `q` / `Ctrl+C` | Dashboard beenden |

## Dashboard-Layout

Das Dashboard zeigt:

- **Profilliste** -- Alle konfigurierten Profile mit Name, Modell, Anbietertyp und Gesundheitsstatus
- **Gesundheitsindikatoren** -- Farbcodierter Status (gruen = gesund, gelb = beeintraechtigt, rot = Circuit Breaker offen)
- **Metrik-Panel** -- Anfragenanzahl, Erfolgsrate und durchschnittliche Latenz pro Profil
- **Protokoll-Panel** -- Echtzeit-scrollende Protokollausgabe

## Gesundheitsstatus

Das Dashboard fuehrt einen Hintergrund-Gesundheitspruefer aus, der alle aktivierten Profile alle ~30 Sekunden anpingt. Der Gesundheitsstatus wird in Echtzeit aktualisiert:

| Farbe | Bedeutung |
|-------|-----------|
| Gruen | Anbieter antwortet normal |
| Gelb | Kuerzliche Fehler erkannt |
| Rot | Circuit Breaker ist offen |
| Grau | Profil ist deaktiviert |
