## Installationsmethoden

<Tabs>
  <TabItem label="Einzeiler" icon="terminal">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
    Das Installationsskript erkennt automatisch Ihr Betriebssystem und die Architektur, laedt die neueste Version von GitHub herunter und installiert nach `~/.local/bin/claudex`.

    Installationsverzeichnis ueberschreiben:
    ```bash
    CLAUDEX_INSTALL_DIR=/usr/local/bin curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
  </TabItem>
  <TabItem label="Cargo" icon="seti:rust">
    ```bash
    cargo install --git https://github.com/StringKe/claudex
    ```
    Erfordert die Rust-Toolchain. Die Binaerdatei wird nach `~/.cargo/bin/claudex` installiert.
  </TabItem>
  <TabItem label="GitHub Releases" icon="github">
    Laden Sie die vorkompilierte Binaerdatei fuer Ihre Plattform von [GitHub Releases](https://github.com/StringKe/claudex/releases) herunter.

    Verfuegbare Zielplattformen:
    - `x86_64-unknown-linux-gnu`
    - `x86_64-unknown-linux-musl`
    - `aarch64-unknown-linux-gnu`
    - `aarch64-unknown-linux-musl`
    - `x86_64-apple-darwin`
    - `aarch64-apple-darwin`
    - `x86_64-pc-windows-msvc`

    Entpacken und die Binaerdatei in Ihren `PATH` legen:
    ```bash
    tar xzf claudex-v*.tar.gz
    chmod +x claudex
    mv claudex ~/.local/bin/
    ```
  </TabItem>
</Tabs>

## Systemvoraussetzungen

| Voraussetzung | Details |
|---------------|---------|
| **Betriebssystem** | macOS (Intel / Apple Silicon), Linux (x86_64 / ARM64), Windows (x86_64) |
| **Claude Code** | Muss installiert und als `claude` im PATH erreichbar sein |
| **Optional** | [Ollama](https://ollama.com) fuer Smart-Routing- und Kontext-Engine-Funktionen |

## Installation ueberpruefen

```bash
claudex --version
```
