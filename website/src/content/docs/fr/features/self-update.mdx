---
title: Mise a jour automatique
description: Gardez Claudex a jour avec une seule commande
---

Claudex peut verifier et installer des mises a jour directement depuis GitHub Releases.

## Verifier les mises a jour

```bash
claudex update --check
```

Cette commande interroge l'API GitHub pour la derniere version et la compare avec votre version installee.

## Installer une mise a jour

```bash
claudex update
```

Cette commande telecharge le dernier binaire de la version pour votre plateforme, remplace le binaire actuel, et verifie l'installation.

## Fonctionnement

1. Interroger l'API GitHub : `https://api.github.com/repos/StringKe/claudex/releases/latest`
2. Comparer la version actuelle avec le tag de la derniere version
3. Telecharger l'archive specifique a la plateforme (par ex. `claudex-v0.2.0-aarch64-apple-darwin.tar.gz`)
4. Extraire et remplacer le binaire en cours d'execution
5. Nettoyer les fichiers temporaires

## Plateformes supportees

Le mecanisme de mise a jour supporte les memes plateformes que les builds de version :

| Plateforme | Cible |
|------------|-------|
| macOS Intel | `x86_64-apple-darwin` |
| macOS Apple Silicon | `aarch64-apple-darwin` |
| Linux x86_64 | `x86_64-unknown-linux-gnu` |
| Linux ARM64 | `aarch64-unknown-linux-gnu` |
| Windows x86_64 | `x86_64-pc-windows-msvc` |
