---
title: Tableau de bord TUI
description: Tableau de bord terminal en temps reel pour la surveillance et la gestion des profils
---

Le tableau de bord TUI fournit une interface terminal en temps reel pour surveiller la sante des profils, visualiser les metriques, et lancer rapidement des sessions Claude Code.

## Lancement

```bash
# Lancement explicite
claudex dashboard

# Lancement automatique (quand des profils existent et qu'aucune sous-commande n'est donnee)
claudex
```

## Raccourcis clavier

| Touche | Action |
|--------|--------|
| `j` / `↓` | Deplacer la selection vers le bas |
| `k` / `↑` | Deplacer la selection vers le haut |
| `Enter` | Lancer Claude Code avec le profil selectionne |
| `t` | Tester la connectivite du profil selectionne |
| `a` | Ajouter un nouveau profil (affiche une indication) |
| `d` | Supprimer le profil selectionne (affiche une indication) |
| `p` | Basculer demarrage/arret du proxy |
| `/` | Entrer en mode recherche |
| `Esc` | Quitter le mode recherche / fermer l'aide |
| `?` | Afficher la fenetre d'aide |
| `q` / `Ctrl+C` | Quitter le tableau de bord |

## Disposition du tableau de bord

Le tableau de bord affiche :

- **Liste des profils** : tous les profils configures avec leur nom, modele, type de fournisseur et etat de sante
- **Indicateurs de sante** : statut code par couleur (vert = sain, jaune = degrade, rouge = disjoncteur ouvert)
- **Panneau de metriques** : nombre de requetes par profil, taux de succes, latence moyenne
- **Panneau de logs** : sortie de logs en defilement en temps reel

## Etat de sante

Le tableau de bord execute un verificateur de sante en arriere-plan qui interroge tous les profils actives toutes les ~30 secondes. L'etat de sante est reflete en temps reel :

| Couleur | Signification |
|---------|--------------|
| Vert | Le fournisseur repond normalement |
| Jaune | Des echecs recents ont ete detectes |
| Rouge | Le disjoncteur est ouvert |
| Gris | Le profil est desactive |
