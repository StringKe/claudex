## Methodes d'installation

<Tabs>
  <TabItem label="Une ligne" icon="terminal">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
    Le script d'installation detecte automatiquement votre OS et votre architecture, telecharge la derniere version depuis GitHub, et installe dans `~/.local/bin/claudex`.

    Changer le repertoire d'installation :
    ```bash
    CLAUDEX_INSTALL_DIR=/usr/local/bin curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
  </TabItem>
  <TabItem label="Cargo" icon="seti:rust">
    ```bash
    cargo install --git https://github.com/StringKe/claudex
    ```
    Necessite la toolchain Rust. Le binaire sera installe dans `~/.cargo/bin/claudex`.
  </TabItem>
  <TabItem label="GitHub Releases" icon="github">
    Telechargez le binaire pre-compile pour votre plateforme depuis les [GitHub Releases](https://github.com/StringKe/claudex/releases).

    Cibles disponibles :
    - `x86_64-unknown-linux-gnu`
    - `x86_64-unknown-linux-musl`
    - `aarch64-unknown-linux-gnu`
    - `aarch64-unknown-linux-musl`
    - `x86_64-apple-darwin`
    - `aarch64-apple-darwin`
    - `x86_64-pc-windows-msvc`

    Extrayez et placez le binaire dans votre `PATH` :
    ```bash
    tar xzf claudex-v*.tar.gz
    chmod +x claudex
    mv claudex ~/.local/bin/
    ```
  </TabItem>
</Tabs>

## Configuration requise

| Prerequis | Details |
|-----------|---------|
| **OS** | macOS (Intel / Apple Silicon), Linux (x86_64 / ARM64), Windows (x86_64) |
| **Claude Code** | Doit etre installe et accessible en tant que `claude` dans votre PATH |
| **Optionnel** | [Ollama](https://ollama.com) pour les fonctionnalites de routage intelligent et de moteur de contexte |

## Verifier l'installation

```bash
claudex --version
```
