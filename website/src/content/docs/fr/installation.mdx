## Méthodes d'installation

<Tabs>
  <TabItem label="Commande unique" icon="terminal">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
    Le script d'installation détecte automatiquement votre OS et votre architecture, télécharge la dernière version depuis GitHub, et l'installe dans `~/.local/bin/claudex`.

    Remplacer le répertoire d'installation :
    ```bash
    CLAUDEX_INSTALL_DIR=/usr/local/bin curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
    ```
  </TabItem>
  <TabItem label="Cargo" icon="seti:rust">
    ```bash
    cargo install --git https://github.com/StringKe/claudex
    ```
    Nécessite la chaîne d'outils Rust. Le binaire sera installé dans `~/.cargo/bin/claudex`.
  </TabItem>
  <TabItem label="GitHub Releases" icon="github">
    Téléchargez le binaire précompilé pour votre plateforme depuis [GitHub Releases](https://github.com/StringKe/claudex/releases).

    Cibles disponibles :
    - `x86_64-unknown-linux-gnu`
    - `x86_64-unknown-linux-musl`
    - `aarch64-unknown-linux-gnu`
    - `aarch64-unknown-linux-musl`
    - `x86_64-apple-darwin`
    - `aarch64-apple-darwin`
    - `x86_64-pc-windows-msvc`

    Extraire et placer le binaire dans votre `PATH` :
    ```bash
    tar xzf claudex-v*.tar.gz
    chmod +x claudex
    mv claudex ~/.local/bin/
    ```
  </TabItem>
</Tabs>

## Configuration requise

| Exigence | Détails |
|----------|---------|
| **OS** | macOS (Intel / Apple Silicon), Linux (x86_64 / ARM64), Windows (x86_64) |
| **Claude Code** | Doit être installé et accessible via la commande `claude` dans votre PATH |
| **Optionnel** | [Ollama](https://ollama.com) pour le routage intelligent et les fonctionnalités du moteur de contexte |

## Vérifier l'installation

```bash
claudex --version
```
