## Installation rapide

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## Comment ça fonctionne

```
claudex run grok
    │
    ├── Démarrer le proxy → 127.0.0.1:13456
    │
    └── exec claude avec :
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

Le proxy traduit de manière transparente entre les protocoles Anthropic et OpenAI, de sorte que Claude Code fonctionne de manière transparente avec tout fournisseur pris en charge. Lors de l'utilisation de profils OAuth, `ANTHROPIC_AUTH_TOKEN` (et non `ANTHROPIC_API_KEY`) est défini pour éviter les conflits avec la connexion par abonnement propre de Claude.
