## Installation rapide

```bash
curl -fsSL https://raw.githubusercontent.com/StringKe/claudex/main/install.sh | bash
```

## Fonctionnement

```
claudex run grok
    │
    ├── Start proxy → 127.0.0.1:13456
    │
    └── exec claude with:
        ANTHROPIC_BASE_URL=http://127.0.0.1:13456/proxy/grok
        ANTHROPIC_AUTH_TOKEN=<session-token>
        ANTHROPIC_MODEL=grok-3-beta
```

Le proxy traduit de maniere transparente entre les protocoles Anthropic et OpenAI, de sorte que Claude Code fonctionne parfaitement avec tout fournisseur supporte. Lors de l'utilisation de profils OAuth, `ANTHROPIC_AUTH_TOKEN` (et non `ANTHROPIC_API_KEY`) est defini pour eviter les conflits avec le mecanisme de connexion par abonnement propre a Claude.
